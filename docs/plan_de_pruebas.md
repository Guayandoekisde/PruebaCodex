## 5) Pruebas mínimas por lenguaje
- **Python:** ejecución de script principal y validación de entradas básicas.
- **C:** compilación exitosa y ejecución con caso nominal.
- **Bash:** ejecución con permisos correctos y manejo de parámetros básicos.
- **ARM64 Assembly:** ensamblado y ejecución de un flujo mínimo (muy pequeño).

> No se requieren frameworks de testing. Se permiten pruebas manuales documentadas.

## 6) Criterios para considerar la práctica terminada
- La documentación está completa y coherente.
- Existe al menos un flujo principal probado.
- Se documentan errores esperados y su manejo.
- El proyecto mantiene alcance pequeño y viable.
